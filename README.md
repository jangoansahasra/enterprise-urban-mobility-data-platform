# Enterprise Urban Mobility Data Platform

An end-to-end data engineering and machine learning platform that ingests customer transaction data, performs ETL, validates data quality, stores data in PostgreSQL, exposes REST APIs with FastAPI, and delivers predictive analytics and business intelligence dashboards.

## Tech Stack

- Python
- PostgreSQL
- SQLAlchemy
- FastAPI
- Docker
- Pandas
- NumPy
- Scikit-Learn
- Git
